# Discriminant
Дискриминант

Решение квадратного уравнения.
            ax²+bx+c=0
    Как найти дискриминант:
            D=b²-4ac
    Корни уравнения:
        > если D<0 - уравнение не имеет корней (нет решений)
        > если D=0 - уравнение имеет 1 корень:
                x=-b²/2a
        > если D>0 - уравнение имеет 2 корня:
                x1=(-b+√D)/(2a)
                x2=(-b-√D)/(2a)
